## 사진 분석 → 게임 배경 개선 체크리스트
1. **필지 불규칙화**: 실제 논은 균일 격자가 아니라 크기·모양 제각각인 필지 모자이크. 필지(여러 셀 묶음) 단위로 논둑을 치고 필지별 색조·수위 미세 차이를 줄 것
2. **배산임수**: 마을 뒤(북쪽)에 숲 우거진 낮은 산 능선. 논은 그 앞 평지에
3. **비닐하우스**: 논 옆 아치형 반투명 하우스 2~3동 — 한국 농촌 시그니처
4. **콘크리트/흙 농로**: 마을~논 사이 길, 논 사이 좁은 둑길
5. **모판 소품**: 모내기철 논둑에 쌓인 연두색 모판
6. **논둑 스케일**: 경계 논둑은 낮고 좁게, 구역 외곽 둑은 높게(1m, 풀 덮임)
7. 전봇대 ✓(구현됨), 정자나무(큰 활엽수 1그루를 마을에)
8. (추후) 이앙기·콤바인 탈것, 소 써레질(역사 모드?)
