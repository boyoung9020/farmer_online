# 농사 시스템 — 벼농사 (사실 모델)

구현: [scripts/paddy.gd](../scripts/paddy.gd), [scripts/rice_field.gd](../scripts/rice_field.gd), [scripts/paddy_panel.gd](../scripts/paddy_panel.gd)

농지(60×40m)를 **4개 논배미(2×2)** 로 나눠 각각을 사실적으로 관리한다.
**논 안에서 E** → 논 관리 패널(상황별 액션 버튼). 노동자는 논을 자동 관리.

## 생육 단계
```
미경작 → 경운(써레질) → [물대기] → 이앙(모내기) → 분얼 → [중간물떼기]
        → 유수형성·출수 → 등숙 → [낙수] → 수확
```

## 관리 요소 → 품질/수확량
- **물(수문)**: 이앙~출수~등숙엔 담수, 분얼 정점엔 중간물떼기, 수확 전 낙수. 시기 틀리면 품질↓.
- **거름 타이밍**: 밑거름(이앙 시) → 분얼비 → 이삭거름. 누락 시 패널티.
- **도복**: 질소 과다(≥60) + 중간물떼기 누락 → 출수 시 도복(품질 ×0.5).
- **병해충**: 분얼·출수기 랜덤 발생, 방치 시 품질↓ → **방제**로 제거.
- **날씨**: 맑음/흐림/비 전역 순환. **비=담수**, **맑음(등숙기)=품질↑**.
- **수확 적기**: 완숙 후 너무 늦으면 과숙(품질↓).
- **품종**: 조생종(빠름·가격↓) / 중만생종(수량·가격↑).

## 수익
`수확 돈 = 셀수 × 7 × 품질 × 품종계수(조생0.9/중만생1.25)`
→ 물·거름·방제·적기를 잘 맞춘 논일수록 더 번다.

## 조작
- 논 안에서 **E**: 관리 패널 (단계에 맞는 버튼: 써레질/물대기/이앙/분얼비/중간물떼기/이삭거름/낙수/수확/방제)
- **농사 노동자**(고용소 E): 논을 돌며 위 과정을 자동 수행
- HUD: `날씨`, `돈`, 상태(도보/트랙터)
