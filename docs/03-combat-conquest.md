# 점령전 / 적 AI

> 최종 타겟은 **멀티플레이**지만, 현재는 **싱글 적 AI**를 먼저 구현.

## 구현됨 (러프 적 AI) — [enemy_manager.gd](../scripts/enemy_manager.gd), [unit.gd](../scripts/unit.gd)
- **적 기지**(북쪽, 어두운 건물+적 깃발)에서 적 병사를 주기적으로 생성(기본 9초, 최대 8명).
- 적 병사는 **플레이어 집결지로 진군**. 도중 아군(노동자/군사)이 탐지 반경(22m) 안에 들면 교전.
- **전투**: 체력 60, 공격 12/0.8초, 사거리 2.2m. 0 이하 사망(`take_damage`→`queue_free`).
- 아군 군사(상점 J/군사)는 집결지에서 대기하다 적 탐지 시 교전. 농사 노동자는 비전투(피격 시 사망).
- 진영/역할은 그룹으로 관리: `player_units/enemy_units`, `player_soldiers/enemy_soldiers` 등.

## 컨셉 (확장 방향)
농사로 번 돈으로 **군사**를 고용해 이웃/적 필지를 공격, 소유권을 빼앗는다.

## 기존 시스템 재사용
- **점령 단위 = 필지(Parcel)**. `parcel.gd`의 `owner_id`(0=중립, 1=플레이어, 2+=적)를 그대로 사용.
- 지적도 로더의 반경/기준점을 키우면 전장 확장.

## 설계 초안
- **용병 고용**: `GameManager.spend_money()`로 비용 지불 → 용병 유닛 스폰.
- **유닛**: 이동/전투 스탯(체력·공격·속도). NavigationServer3D로 길찾기.
- **공격 판정**: 적 필지에 아군 용병이 일정 시간 점유 → `owner_id` 이전.
- **방어**: 적도 용병 보유, 충돌 시 전투.
- **승패**: 모든 필지 점령 또는 목표 비율 달성.

## 선행 작업
1. 유닛 베이스(CharacterBody3D + 상태머신: 이동/교전/사망).
2. 필지 소유권 시각화(색/깃발) + 소유권 변경 이벤트.
3. 간단한 적 AI(가까운 중립/아군 필지로 진군).
4. 고용 UI.
